Projeto de Power BI

Projeto de dash de vendas

Este projeto tem o intuito de aplicar alguns metodos para análise de dados, o uso de conhecimentos no front-end e back-end, a aplicação de elementos gráficos visuais e controle de usuários.

Acesse o App

Django

Django é usado como framework de desenvolvimento web, escrito em Python, utilizando o padrão model-template-view.

Bootstrap

Bootstrap foi escolhido como framework para o front-end pela facilidade em sua aplicação de componentes e responsividade.

Heroku

Heroku é a plataforma de nuvem como serviço atendendo várias linguagens de programação, custos e suporte amigavel com aplicações estudantis.

PostgreSQL

PostgreSQL é o sistema gerenciador de banco de dados objeto relacional disponibilizado pela plataforma do Heroku.
